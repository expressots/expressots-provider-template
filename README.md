## About

Scaffold for providers that integrate with ExpressoTS through the System Plugin Pattern. Includes module layout, provider registration, and publishing defaults.

## Quick Start

```bash
git clone https://github.com/expressots/expressots-provider-template.git my-provider
cd my-provider
npm install
npm run build
```

Customize the provider implementation, then publish under your scoped npm package name.
